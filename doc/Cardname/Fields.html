Overview

Name-based “Fields” are compound names in which the right name is treated as an attribute of the left. (Eg MyName+address is a field of MyName)

def field_name tag
  tag = tag.to_s[1..-1] if !tag.is_a?(Symbol) && tag.to_s[0] == "+"
  [self, tag].to_name
end

def field_of? context
  return false unless compound?

  if context.present?
    absolute_name(context).left_name.key == context.to_name.key
  else
    s.match?(/^\s*\+[^+]+$/)
  end
end
