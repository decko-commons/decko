Overview

The Content::Parser breaks content strings into an Array of “chunks”, each of which may be an instance of a Chunk class or a simple String.

def parse content
  @content = content
  @chunks = []
  return @chunks unless content.is_a? String

  @position = @last_position = 0
  @interval_string = ""
  parse_chunks
  @chunks
end
